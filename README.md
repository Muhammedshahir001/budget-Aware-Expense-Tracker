# Budget Aware Expense Tracker

This is a MERN stack  (backend + frontend) to track expenses and budgets.

## Quick start

1. Backend
   - cd server
   - npm install
   - copy .env.example to .env and fill MONGO_URI and JWT_SECRET
   - npm run dev

2. Frontend
   - cd client
   - npm install
   - npm run dev
   - open http://localhost:5173

API base: http://localhost:5000/api
